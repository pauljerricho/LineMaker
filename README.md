# LineMaker 사용법

## 개요
LineMaker은 점들을 입력받아 최근접 이웃 경로 알고리즘으로 모든 점을 하나의 직선 경로로 연결하는 Windows Forms 애플리케이션입니다.

## 주요 기능
- 점 개수 설정 및 점 입력
- 최근접 이웃 알고리즘을 통한 자동 경로 계산
- 시각적인 점과 선 표시
- 초기화 기능

## 사용 방법

### 1. 점 개수 설정
1. 화면 상단의 텍스트박스에 원하는 점의 개수를 입력합니다.
2. **"점 개수 설정"** 버튼을 클릭합니다.
3. 상태 레이블에 입력해야 할 점의 개수가 표시됩니다.

### 2. 점 입력
1. 점 개수 설정 후, 화면의 원하는 위치를 마우스 왼쪽 버튼으로 클릭합니다.
2. 클릭한 위치에 빨간색 점이 표시됩니다.
3. 상태 레이블에 현재 입력된 점의 개수가 표시됩니다 (예: "현재: 3/5").
4. 설정한 개수만큼 점을 모두 입력하면 입력 모드가 종료됩니다.

### 3. 연결 계산
1. 모든 점 입력이 완료되면 **"연결 계산"** 버튼이 활성화됩니다.
2. **"연결 계산"** 버튼을 클릭합니다.
3. 최근접 이웃 알고리즘이 실행되어 모든 점이 파란색 선으로 연결됩니다.
4. 상태 레이블에 연결된 선의 개수가 표시됩니다.

### 4. 초기화
- **"초기화"** 버튼을 클릭하면 모든 점과 선이 제거되고 프로그램이 초기 상태로 돌아갑니다.

## UI 구성 요소

### 버튼
- **점 개수 설정**: 입력할 점의 개수를 설정합니다.
- **연결 계산**: 모든 점을 최근접 이웃 알고리즘으로 연결합니다.
- **초기화**: 모든 데이터를 초기화합니다.

### 텍스트박스
- 점 개수를 입력하는 필드입니다.

### 상태 레이블
- 현재 프로그램의 상태와 진행 상황을 표시합니다.

## 알고리즘 설명

### 최근접 이웃 경로 알고리즘
1. 첫 번째 점을 시작점으로 선택합니다.
2. 현재 점에서 가장 가까운 미방문 점을 찾습니다.
3. 두 점을 선으로 연결합니다.
4. 찾은 점을 현재 점으로 설정하고 방문 목록에 추가합니다.
5. 모든 점을 방문할 때까지 2-4단계를 반복합니다.

이 알고리즘은 모든 점을 하나의 연속된 경로로 연결하며, 각 단계에서 가장 가까운 점을 선택하여 전체 경로의 길이를 최소화합니다.

## 주의사항
- 최소 2개 이상의 점이 필요합니다.
- 점 개수는 양의 정수만 입력 가능합니다.
- 점 입력 모드에서는 설정한 개수만큼만 입력할 수 있습니다.
